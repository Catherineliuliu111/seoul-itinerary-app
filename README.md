# Seoul 3-Day Itinerary

A mobile-first Seoul travel itinerary page built with React and Vite.

## Highlights

- Three-day itinerary with collapsible timeline cards.
- Restaurant, cafe, shopping, photobooth, and transport recommendations.
- Day 2 includes Milestone Coffee Roasters Seongsu for Vienna Coffee / Einspänner.
- Optimized for phone-width viewing, similar to a consumer itinerary app.

## Run Locally

```bash
cd frontend
npm install
npm run dev -- --host 127.0.0.1
```

Open `http://127.0.0.1:5173/`.

## Build

```bash
cd frontend
npm run build
```
